Here's a list of common IT architecture documents and their corresponding KPIs (Key Performance Indicators):

    Enterprise Architecture Document
        KPI: % of IT projects aligned with business objectives
        KPI: Return on Investment (ROI) of IT initiatives
    Technical Architecture Document
        KPI: System uptime (%)
        KPI: Average response time (ms)
    Application Architecture Document
        KPI: Application availability (%)
        KPI: # of defects per release
    Data Architecture Document
        KPI: Data quality score (%)
        KPI: Data storage utilization (%)
    Security Architecture Document
        KPI: # of security incidents
        KPI: % of systems compliant with security policies
    Network Architecture Document
        KPI: Network uptime (%)
        KPI: Average network latency (ms)
    Infrastructure Architecture Document
        KPI: Server utilization (%)
        KPI: # of incidents due to infrastructure issues
    Integration Architecture Document
        KPI: # of integrations completed on time
        KPI: % of APIs meeting SLAs
    Solution Architecture Document
        KPI: Solution adoption rate (%)
        KPI: # of issues resolved within SLA
    Roadmap and Strategy Document
        KPI: % of projects meeting deadlines
        KPI: % of projects within budget
